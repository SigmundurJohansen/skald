# Skald

### How to build

 ```
g++ --std=c++17 -o skald main.cpp src/file_system.cpp src/makefile.cpp external/yyjson.c 
 ```

only works on windows at the moment

### How to use

run it in project folder
 ```
skald
 ```
build running
 ```
make
 ```
and to clean run make clean
 ```
make clean
 ```
